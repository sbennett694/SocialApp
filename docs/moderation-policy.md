# Moderation Policy (V1)

## Core Rule
Political content is not allowed on this platform.

## Enforcement (V1)
- All posts and comments are checked before publish.
- Content with political keywords/phrases is rejected.
- API returns a clear rejection reason and moderation code.

## Planned Enhancements (Post-V1)
- Context-aware NLP classifier for political intent.
- OCR-based moderation for image text.
- Human appeal queue for false positives.
